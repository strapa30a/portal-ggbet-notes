# portal-ggbet-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。页面内容主要涉及通用信息展示、工具页面、静态资料存档等，不针对任何特定网站或服务。本仓库仅作为静态页面的托管与版本管理用途。

## 目录说明

```
.
├── docs/            # 静态页面源文件
├── pages/           # 已构建或归档的独立 HTML 页面
├── assets/          # 页面引用的公共资源（样式、脚本、图片等）
├── README.md        # 本文件
└── LICENSE          # 开源许可证
```

- `docs/`：存放可编辑或正在开发中的 HTML 页面源文件。
- `pages/`：存放已完成归档、可直接访问的独立 HTML 页面。
- `assets/`：存放页面所需的公共静态资源，如 CSS、JavaScript、图像文件等。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容以信息展示、工具界面或静态资料存档为主，不包含交互性后台逻辑。
- 所有页面均经过基本清理与格式化，确保在主流浏览器中可直接打开使用。
- 归档页面可能不定期更新，但不会主动引入外部跟踪或分析代码。

## 维护说明

- 本仓库由维护者不定期更新，不接受外部直接推送页面内容。
- 如发现页面显示异常或有明显内容错误，欢迎通过 Issues 提交反馈。
- 本仓库不提供任何形式的担保或承诺，页面内容仅供参考与存档用途。
- 所有页面均遵循仓库附带的许可证条款，详情请参阅 LICENSE 文件。
